Quando usar o Get? Quando usar o Post?

Para o uso do Get, a avaliação que se faz é se os dados pedidos não são sensíveis. Por exemplo: Nome, Peso, idade, altura...

O contrário se faz com o uso do Post. Quando os dados são extremamente sensíveis. Por exemplo: senha, número do cartão de crédito/débito...
